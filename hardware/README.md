# Hardware

This directory contains all hardware design files for the Sesame Robot.

## Directory Structure

- **cad/** - 3D CAD models and STL files for 3D printing
- **schematics/** - Electronic schematics and circuit diagrams
- **pcb/** - PCB design files (if applicable)
- **bom/** - Bill of Materials (BOM) spreadsheets and parts lists
- **manufacturing/** - Manufacturing files (Gerbers, drill files, cut sheets)

## Hardware Requirements

See the main README.md for detailed hardware requirements including:
- Microcontroller (Arduino-compatible AVR board)
- 8x MG90 Servo Motors
- PCA9685 16-Channel PWM/Servo Driver
- 0.96" I2C OLED Display (SSD1306)
- 5V Power Supply (2-3A)
- 3D Printed Parts

## Fabrication Notes

- Material: PLA / PLA+
- Infill: 15-20%
- Supports: Minimal (check individual file notes)
- Most parts designed to print without supports
